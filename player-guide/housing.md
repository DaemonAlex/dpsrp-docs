# Housing Guide

Everything you need to know about owning property in Los Santos.

## Overview

The housing system (qs-housing) allows you to purchase and customize your own home in the city.

## Finding a Property

### Real Estate Agent
- Visit Dynasty 8 Real Estate
- Browse available properties
- View property details and pricing

### Property Types
| Type | Price Range | Features |
|------|-------------|----------|
| Apartments | $50K - $200K | Basic storage, small |
| Houses | $200K - $500K | Garage, more space |
| Mansions | $500K+ | Multiple garages, large |

## Purchasing

1. Find a property you like
2. Check if it's available (not owned)
3. Have enough money in bank
4. Complete the purchase
5. Receive keys to your new home

## Property Features

### Storage
- Personal stash for items
- Safe storage (items persist)
- Wardrobe for outfit changes

### Garage
- Store personal vehicles
- Number of slots varies by property
- Access from property menu

### Customization
- Furniture placement
- Decoration options
- Upgrades available

## Managing Your Property

### Access
- Use keys to enter (item in inventory)
- Give keys to trusted players
- Lock/unlock doors

### Selling
- List property for sale
- Set asking price
- Transfer to new owner

## Tips

1. **Location matters** - Consider proximity to work
2. **Check garage size** - Make sure it fits your vehicles
3. **Budget wisely** - Don't spend all your money on housing
4. **Secure your home** - Lock doors when leaving
