# Controls & Keybinds

![Using Phone](../images/phone/using%20phone.jpg)

*Master the controls and you'll master the city*

Complete list of keyboard controls for DPSRP.

## General Controls

| Key | Action |
|-----|--------|
| W/A/S/D | Movement |
| Shift | Sprint |
| Space | Jump |
| Ctrl | Crouch/Stealth |
| E | Interact |
| G | Radial Menu |
| Tab | Scoreboard |

## UI & Menus

| Key | Action |
|-----|--------|
| F1 | Phone |
| F2 | Inventory |
| F5 | Emote Menu |
| F6 | Job Menu (when on duty) |
| F7 | Invoice/Billing Menu |
| F10 | Help/Guidebook |
| M | Map |
| T | Chat |
| Escape | Pause Menu |

## Vehicle Controls

| Key | Action |
|-----|--------|
| F | Enter/Exit Vehicle |
| W/S | Accelerate/Brake |
| A/D | Steer |
| Space | Handbrake |
| H | Headlights |
| Q | Left Indicator |
| E | Right Indicator |
| G | Seatbelt |
| B | Point at location |
| X | Roll down window |

## Combat & Weapons

| Key | Action |
|-----|--------|
| Tab (hold) | Weapon Wheel |
| Left Mouse | Fire/Attack |
| Right Mouse | Aim |
| R | Reload |
| X | Hands Up |
| Caps Lock | Ragdoll |

## Communication

| Key | Action |
|-----|--------|
| N | Push to Talk (Voice) |
| ` (Grave) | Voice Range Cycle |
| T | Text Chat |
| Y | /me command |
| Numpad + | Radio PTT |

## Target System (ox_target)

| Key | Action |
|-----|--------|
| Left Alt (hold) | Show interaction targets |
| Left Mouse | Select target option |

## Emotes & Animations

| Key | Action |
|-----|--------|
| F5 | Emote Menu |
| ] | Cancel Animation |

## Job-Specific Controls

### Police/EMS
| Key | Action |
|-----|--------|
| F6 | Duty Menu |
| Numpad 4 | Toggle Lights |
| Numpad 5 | Toggle Siren Mode |
| E (on suspect) | Cuff/Uncuff |

### Mechanic
| Key | Action |
|-----|--------|
| F6 | Mechanic Menu |
| E (on vehicle) | Repair options |

### Pilot (Airlines)
| Key | Action |
|-----|--------|
| F6 | Pilot Menu |
| Shift | Throttle Up |
| Ctrl | Throttle Down |
| Numpad 8/5 | Pitch |
| Numpad 4/6 | Roll |

## Voice Chat Ranges

Cycle with ` (Grave key):
1. **Whisper** - 3m range
2. **Normal** - 8m range
3. **Shout** - 15m range

## Quick Commands

| Command | Action |
|---------|--------|
| /help | Open guidebook |
| /me [action] | RP action text |
| /do [description] | RP description |
| /ooc [message] | Out of character chat |
| /report [message] | Report to staff |
| /gps [location] | Set GPS destination |
| /id | Show your ID |
